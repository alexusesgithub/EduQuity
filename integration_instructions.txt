CAREER GUIDANCE CHATBOT INTEGRATION INSTRUCTIONS

1. ADD TO YOUR EXISTING HTML FILE:
   Add this line before the closing </body> tag:
   <script src="career_chatbot.js"></script>

2. YOUR EXISTING CHAT BUTTON:
   The chatbot will automatically work with buttons that have these IDs/classes:
   - id="chat-assistant"
   - class="chat-assistant"
   - data-chat attribute
   - class="chatbot-btn"
   - id="chatbot-button"

3. IF YOUR BUTTON IS DIFFERENT:
   Either:
   - Add id="chat-assistant" to your existing button
   - Or modify line 120 in career_chatbot.js to match your button selector

4. FEATURES INCLUDED:
   ✓ Maintains conversation context
   ✓ Professional career counselor responses
   ✓ Student-focused guidance
   ✓ Quick suggestion buttons
   ✓ Responsive design
   ✓ No backend required

5. CUSTOMIZATION:
   - Edit responses in the initializeResponses() method
   - Modify styling in addChatbotStyles() method
   - Add more keywords as needed
